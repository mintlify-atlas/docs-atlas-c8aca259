**TSGAM Estimator** is a Python library for time series load forecasting built on Generalized Additive Models (GAMs). It provides a flexible, interpretable modeling framework that decomposes time series into seasonal patterns, exogenous variable effects, trends, and residual structure — all fitted via regularized convex optimization.

The library follows the scikit-learn API (`fit` / `predict`), making it easy to integrate into existing data science workflows.

## Why TSGAM?

Most forecasting libraries are either black-box neural networks or rigid statistical models. TSGAM sits in between: it is fully interpretable (each component has a direct physical meaning), yet flexible enough to capture complex real-world patterns.

## Model overview

A TSGAM model is an additive combination of independent components:

```
ŷ(t) = constant
      + Fourier(t)          # seasonal patterns
      + Σ spline(xᵢ, t)    # exogenous variables (e.g. temperature)
      + trend(t)            # long-term drift
      + outlier(t)          # anomalous periods
      + AR(residuals)       # autoregressive correction
```

Each component is optional and independently configured. You can start with just a Fourier seasonal model and progressively add complexity.
