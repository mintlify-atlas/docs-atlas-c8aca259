**TSGAM Estimator** is an interpretable, scikit-learn-compatible Python library for time series forecasting. It decomposes your time series into seasonal patterns, exogenous variable effects, trends, and residual structure — all fitted via regularized convex optimization.

## Key features

## How it works

A TSGAM model additively combines independent components:

```python
from tsgam_estimator import (
    TsgamEstimator, TsgamEstimatorConfig,
    TsgamMultiPeriodicConfig, TsgamSplineConfig,
    PERIOD_HOURLY_DAILY, PERIOD_HOURLY_WEEKLY, PERIOD_HOURLY_YEARLY,
)

# Configure seasonal and exogenous components
config = TsgamEstimatorConfig(
    multi_periodic_config=TsgamMultiPeriodicConfig(
        num_harmonics=[6, 4, 3],
        periods=[PERIOD_HOURLY_DAILY, PERIOD_HOURLY_WEEKLY, PERIOD_HOURLY_YEARLY],
    ),
    exog_config=[TsgamSplineConfig(n_knots=10, lags=[-1, 0, 1])],
)

# Fit and predict — standard scikit-learn API
estimator = TsgamEstimator(config=config)
estimator.fit(X_train, y_train)
y_pred = estimator.predict(X_test)
```
