## Installation

TSGAM Estimator is distributed as the `tsgam_estimator` Python package. It requires Python 3.12 or later.

<CodeGroup>
```bash uv
uv add tsgam_estimator
```

```bash pip
pip install tsgam_estimator
```
</CodeGroup>

To install the development version directly from the repository:

```bash
uv sync
```

## Fit your first model

The following example fits a seasonal GAM to one year of synthetic hourly data using daily, weekly, and yearly Fourier components.

<Steps>
  <Step title="Import and configure">
    ```python
    import numpy as np
    import pandas as pd
    from tsgam_estimator import (
        TsgamEstimator,
        TsgamEstimatorConfig,
        TsgamMultiPeriodicConfig,
        PERIOD_HOURLY_DAILY,
        PERIOD_HOURLY_WEEKLY,
        PERIOD_HOURLY_YEARLY,
    )

    # Seasonal configuration: daily, weekly, and yearly patterns
    multi_periodic = TsgamMultiPeriodicConfig(
        num_harmonics=[6, 4, 3],
        periods=[PERIOD_HOURLY_DAILY, PERIOD_HOURLY_WEEKLY, PERIOD_HOURLY_YEARLY],
    )

    config = TsgamEstimatorConfig(
        multi_periodic_config=multi_periodic,
        exog_config=None,  # no exogenous variables yet
    )
    ```
  </Step>

  <Step title="Prepare data">
    `X` must be a `pandas.DataFrame` with a `DatetimeIndex`. The target `y` is a 1-D NumPy array.

    ```python
    # One year of hourly data
    dates = pd.date_range("2020-01-01", periods=8760, freq="h")
    X = pd.DataFrame(index=dates)  # no exogenous columns needed for pure seasonal model

    # Synthetic load with daily + weekly seasonality and noise
    t = np.arange(8760)
    y = (
        50
        + 10 * np.sin(2 * np.pi * t / 24)          # daily
        + 5  * np.sin(2 * np.pi * t / (24 * 7))    # weekly
        + np.random.normal(0, 1, 8760)
    )
    ```
  </Step>

  <Step title="Fit and predict">
    ```python
    estimator = TsgamEstimator(config=config)
    estimator.fit(X, y)

    # Predict on a future window
    future_dates = pd.date_range("2021-01-01", periods=168, freq="h")
    X_future = pd.DataFrame(index=future_dates)
    y_pred = estimator.predict(X_future)

    print(y_pred[:5])
    ```
  </Step>
</Steps>

## Add an exogenous variable

Real load forecasting almost always benefits from weather inputs. Add a temperature variable with a cubic spline basis and lead/lag terms:

```python
from tsgam_estimator import TsgamSplineConfig

# Temperature modeled with a cubic spline (10 knots) and +/-1 hour lead/lag
spline_cfg = TsgamSplineConfig(n_knots=10, lags=[-1, 0, 1])

config = TsgamEstimatorConfig(
    multi_periodic_config=multi_periodic,
    exog_config=[spline_cfg],
)

# X must have one numeric column per exog_config entry
temperature = 20 + 10 * np.sin(2 * np.pi * t / (24 * 365)) + np.random.normal(0, 2, 8760)
X = pd.DataFrame({"temperature": temperature}, index=dates)

estimator = TsgamEstimator(config=config)
estimator.fit(X, y)
```

## Use `get_recommended_periods`

If you don't know which periods to use, let the library infer them from your data:

```python
from tsgam_estimator import get_recommended_periods

periods, harmonics = get_recommended_periods(X, include_harmonics=True)

multi_periodic = TsgamMultiPeriodicConfig(
    num_harmonics=harmonics,
    periods=periods,
)
```
